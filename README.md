# THE RETRO CENTER 🕹️

デスクトップの片隅で **AI が永遠にゲームを解き続ける**レトロゲームウィジェット集です。
各ゲームはグロー・グラデーション・パーティクル等のモダンな演出で描画します。
メニューバー（トレイ）にアーケード筐体アイコンと「THE RETRO CENTER」を表示します。
2048・ぷよぷよ・Rush Hour・インベーダー・ボンバーマン・テトリス・スネーク・ライフゲーム・ブロック崩し・タワーディフェンス・ヒーローのくせに生意気だ をセレクトボックスで切り替えられます。

## ダウンロード

[**Releases ページ**](../../releases/latest) から OS に合ったファイルをダウンロードしてください。

| OS | ファイル |
|---|---|
| macOS (Apple Silicon) | `DeskWidgets-x.x.x-arm64.dmg` |
| macOS (Intel) | `DeskWidgets-x.x.x.dmg` |
| Windows | `DeskWidgets-Setup-x.x.x.exe`（インストーラ）または `DeskWidgets-x.x.x-portable.exe`（インストール不要） |

> **macOS で「開発元を検証できません」と出る場合**（未署名のため）:
> アプリを右クリック →「開く」で起動できます。それでも開けない場合はターミナルで
> `xattr -cr /Applications/DeskWidgets.app` を実行してください。
>
> **Windows で SmartScreen の警告が出る場合**:「詳細情報」→「実行」で起動できます。

## 機能

### ゲームウィジェット（画面右下に常駐）

- **AI 自動運転**: 起動した瞬間から AI がプレイし続け、ゲームオーバーになっても自動リスタートで永遠に動き続けます
- **3 つのゲームをボタンで切替**
  - **2048** — Expectimax 探索。2048 を超えても無限に継続（4096/8192/16384/32768 専用色）
  - **ぷよぷよ** — 8×15 のやや大きめ盤面。AI は常に「自己最大連鎖の更新」を狙って土台を組み上げ、
    記録を超える発火だけを優先（記録未満は我慢）。死なない範囲で大連鎖を伸ばす（シミュ最大17連鎖）。
    丸く艶やかなスライム風ドット絵ぷよ（光沢・大きな目・影、同色は隣接で連結）。消去エフェクト＋スプリングでぷるっと潰れる着地＋連鎖演出。
    **最大連鎖を記録**し、フッターの「▶ﾘﾌﾟﾚｲ」でその盤面を再現再生できる
  - **Rush Hour** — BFS で最短手順を解き、クリアするたびに新しい問題を自動生成。
    車の台数（6〜12 台）をフッターの −/＋ で調整可能
  - **インベーダー** — AI が敵弾を予測回避しつつ最下段の敵から撃ち落とす。ウェーブを全滅させると
    レベルアップし、敵の数・速度・弾が増えてどんどん難しくなる。全滅すると Lv.1 から自動リスタート
  - **ボンバーマン** — 4 体のドット絵ロボット AI が自動で爆弾を置き合う 4 人対戦。爆風の到達時間を読んで退避し、
    逃げ道がある時だけ爆弾を設置。一定時間でブロックが外周から積まれて盤面が狭まる（サドンデス）。
    最後の 1 体になると次ラウンドを自動開始。AI 自動運転時のみ高速化、手動操作時は通常速度。
    フッターの −/＋ でブロック密度を調整、ウィジェットをドラッグで広げると盤面のマス数が増える（最大 17×19）
  - **テトリス** — AI が全配置（回転×位置）を評価して最善手を選ぶ（集約高さ・消去ライン・穴・凸凹で評価）。
    ライン消去でレベルが上がり落下が速くなる。ゴースト＆ NEXT 表示、積み切ると自動リスタート
  - **スネーク** — 大きめのドット絵盤面。AI は餌への最短路を取りつつ「進んだ先に十分な空間が残る時だけ」進み、
    危険なら最も広い空間へ逃げて自滅を避ける。餌を食べるほどレベルが上がり速くなる
  - **ライフゲーム** — Conway のライフゲーム（トーラス・ドット絵）。初期値はランダムで、停止・全滅・
    **周期1/2のループ（点滅で止まる状態）**・長く続く低活性を検出するとランダムに細胞を足して動きを復活。
    フッターのセレクトで有名プリセット（グライダー / グライダー銃 / パルサー / R-ペントミノ / どんぐり /
    LWSS / ペンタデカスロン）をセット可能。「＋ﾗﾝﾀﾞﾑ」ボタン（手動）でいつでもランダム追加
  - **ブロック崩し** — AI がパドルでボールを受け、全消しでレベルアップ（速く・多く・硬く）。
    開始レベルをフッターの −/＋ で指定でき、Lv5 以降はステージをランダム生成。
    アイテム（分裂 / ボール巨大化 / パドル拡大 / 残機+ / スロー）と、残像・グロー・パーティクル等のモダン演出付き
  - **タワーディフェンス** — 細かいマス目の盤面を蛇行する道に、AI が所持金で 8 種の兵器
    （アロー / キャノン(範囲) / フロスト(減速オーラ) / レーザー(ビーム) / スナイパー(長射程) /
    テスラ(連鎖) / ミサイル(誘導範囲) / ポイズン(継続ダメージ)）を要所に建設・強化して自動防衛。
    AI は構成比を意識して多彩な兵器を揃え、高級兵器は貯金して購入。ウェーブを重ねるほど敵が増え・
    固く・速くなり、5 の倍数でボスが出現。ライフ 0 で自動リスタート。グラデ地形・回転する発光砲塔・
    弾／ビーム／連鎖／パーティクル等のドット絵＋モダン演出付き
  - **ヒーローのくせに生意気だ** — 『勇者のくせになまいきだ。』の仕様を調査して反映したダンジョン生態系シム。
    AI は破壊神となり**つるはしで岩を掘る（掘削力を消費）だけ**で、湧いたモンスターは命令できず**自律する生態系**として振る舞う。
    土には 2 種の資源が宿る：**養分**（総量保存・新規流入はヒーローの死のみ）と**魔分**（初期ゼロ・ヒーローの詠唱/死亡でのみ発生）。
    **運搬役**（ニジリゴケ＝養分／エレメント＝魔分）が資源を吸って別の土へ吐き出し“濃縮”し、濃い土から上位種が湧く（定石）。
    2 系統の食物連鎖〔養分系：ニジリゴケ→ガジガジムシ→トカゲおとこ／魔力系：エレメント→リリス→**ドラゴン**（希少な頂点）〕で
    捕食・繁殖し、死んだ者は資源を周囲へ飛散（保存則）。侵入は**少数精鋭**（1 回 1〜3 人＋節目に勇者/大勇者）で、数より
    個の強さが日々上がる。ヒーローが通路を辿って**魔王を掴み、入口へ運び出す**と GAME OVER（魔王は戦わず・運搬中は鈍足なので
    道中で討てば奪還）。負けたら自動リスタート。AI は魔分を溜めて**ドラゴンを育てる**ことを目指す
- **手動モード**: ウィジェットをクリックしてフォーカスすると AI が止まり、矢印キーで操作できます。
  フォーカスを外すと AI が再開（Rush Hour は車をクリックで選択 → 矢印キー）
- **「＋」ボタン**でウィジェットを複数並べられます（右下から左へ整列）
- **ゲーム選択**: ヘッダーのセレクトボックスで各ゲームから直接選べます
- **効果音**: 各ゲームに合わせた効果音（Web Audio 生成）。ヘッダーの 🔊/🔇、トレイメニュー、または
  ショートカット **⌘/Ctrl + Shift + M** で**全ウィジェット一括** ON/OFF。**一括非表示中は自動でミュート**（再表示で復帰）
- **一括非表示と復帰**: ヘッダーの「—」ボタンで全ウィジェットを一括で隠せます。復帰は次のいずれかで:
  - グローバルショートカット **⌘/Ctrl + Shift + H**（取得済みでない場合は ⌥+H 等にフォールバック）
  - **メニューバーのトレイアイコン**（左クリックで復帰、メニューから「すべて表示」「新しいウィジェットを追加」「終了」）
  - **Dock アイコン**をクリック（macOS）
- **前面 / 背面の切替**: トレイメニューの「前面に表示」「背面に表示」、またはショートカット **⌘/Ctrl + Shift + B**。
  背面は「常に最前面」を解除して他ウィンドウの後ろ（デスクトップ寄り）に置く挙動です
- ウィンドウ端のドラッグでリサイズ（縦横比固定）、ヘッダーのドラッグで移動
- スコア・周回数・最大連鎖などは自動保存

## 開発者向け

```bash
git clone https://github.com/nmatsumoto4/desk-widgets.git
cd desk-widgets
npm install
npm start          # ビルド（esbuild）してから起動
npm run build      # src/*.ts → ルートの *.js を生成
npm run build:watch # 変更を監視して自動ビルド
npm run typecheck  # tsc による型チェック（noEmit）
npm run dist       # ビルドして配布用パッケージを出力（dist/）
```

### TypeScript / ビルド

ソースはすべて **TypeScript**（`src/*.ts`）です。`esbuild`（`build.mjs`）が各ファイルを
個別トランスパイルし、`index.html` が読み込むルートの `*.js`（および `main.js` / `preload.js`）を生成します。
生成された `*.js` はビルド成果物のため `.gitignore` 済みです（コミットするのは `src/*.ts`）。

- レンダラ各スクリプトは ESM/CJS の import を使わず `window.*` グローバルで連携します（`src/globals.d.ts` に型定義）
- 型チェックは `tsconfig.json`（レンダラ=DOM）と `tsconfig.main.json`（Electron メイン=Node）の2系統
- 移行版のため `noImplicitAny` / `strictNullChecks` は緩和、その他の strict 検査は有効

### ファイル構成

| ファイル | 役割 |
|---|---|
| `build.mjs` / `tsconfig*.json` | esbuild ビルドスクリプト / 型チェック設定 |
| `src/main.ts` / `src/preload.ts` | Electron メイン（ウィンドウ生成・整列・IPC） |
| `src/globals.d.ts` | `window.*` グローバルと共通インターフェース（`WidgetCtx` / `WidgetModule`）の型定義 |
| `index.html` / `style.css` / `src/app.ts` | ゲームウィジェット共通シェル（モード切替・手動/AI 切替） |
| `src/game.ts` / `src/ai.ts` / `src/mod2048.ts` | 2048 ロジック / Expectimax AI / 描画 |
| `src/puyo.ts` / `src/puyo-ai.ts` / `src/mod-puyo.ts` | ぷよぷよ ロジック / 連鎖 AI / 描画 |
| `src/rush.ts` / `src/mod-rush.ts` | Rush Hour ロジック・BFS ソルバー・問題生成 / 描画 |
| `src/mod-invaders.ts` | インベーダー（ゲーム進行・回避/掃討 AI・レベル管理・描画） |
| `src/mod-bomber.ts` | ボンバーマン（4 体 AI 対戦・爆風危険マップ/退避 AI・ブロック調整・描画） |
| `src/mod-tetris.ts` | テトリス（配置評価 AI・レベル/落下速度・ゴースト/NEXT・描画） |
| `src/mod-snake.ts` | スネーク（最短路＋空間確保 AI・可変大グリッド・ドット絵描画） |
| `src/mod-life.ts` | ライフゲーム（トーラス B3/S23・停滞検出ランダム注入・有名プリセット・ドット絵） |
| `src/mod-breakout.ts` | ブロック崩し（受け/狙い撃ち AI・可変グリッド・アイテム・モダン演出） |
| `src/mod-td.ts` | タワーディフェンス（8 種兵器・蛇行路生成・構成比/要所優先の建設 AI・ウェーブ難化・ドット絵） |
| `src/mod-hero.ts` | ヒーローのくせに生意気だ（発掘で育てる 2 系統の食物連鎖生態系・魔王の拉致防衛・距離場誘導 AI・ドット絵） |

新しいゲームは `show/hide/setAuto/key/relayout` の共通インターフェース（`WidgetModule`）に載せるだけで追加できます。
