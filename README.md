# THE RETRO CENTER 🕹️

デスクトップの片隅で **AI が永遠にゲームを解き続ける**レトロゲームウィジェット集です。
メニューバー（トレイ）にアーケード筐体アイコンと「THE RETRO CENTER」を表示します。
2048・ぷよぷよ・Rush Hour・インベーダー・ボンバーマン・テトリス・スネーク・ライフゲーム をセレクトボックスで切り替えられます。

## ダウンロード

[**Releases ページ**](../../releases/latest) から OS に合ったファイルをダウンロードしてください。

| OS | ファイル |
|---|---|
| macOS (Apple Silicon) | `DeskWidgets-x.x.x-arm64.dmg` |
| macOS (Intel) | `DeskWidgets-x.x.x.dmg` |
| Windows | `DeskWidgets-Setup-x.x.x.exe`（インストーラ）または `DeskWidgets-x.x.x-portable.exe`（インストール不要） |

> **macOS で「開発元を検証できません」と出る場合**（未署名のため）:
> アプリを右クリック →「開く」で起動できます。それでも開けない場合はターミナルで
> `xattr -cr /Applications/DeskWidgets.app` を実行してください。
>
> **Windows で SmartScreen の警告が出る場合**:「詳細情報」→「実行」で起動できます。

## 機能

### ゲームウィジェット（画面右下に常駐）

- **AI 自動運転**: 起動した瞬間から AI がプレイし続け、ゲームオーバーになっても自動リスタートで永遠に動き続けます
- **3 つのゲームをボタンで切替**
  - **2048** — Expectimax 探索。2048 を超えても無限に継続（4096/8192/16384/32768 専用色）
  - **ぷよぷよ** — 8×15 のやや大きめ盤面で大連鎖を狙う AI（実測：死亡 0・最大 8 連鎖）。
    ドット絵のぷよ（大きな目・ツヤ・影、同色は隣接でつながって見える本家風）。消去・ぬるっと落下・連鎖の演出付き。
    **最大連鎖を記録**し、フッターの「▶ﾘﾌﾟﾚｲ」でその盤面を再現再生できる
  - **Rush Hour** — BFS で最短手順を解き、クリアするたびに新しい問題を自動生成。
    車の台数（6〜12 台）をフッターの −/＋ で調整可能
  - **インベーダー** — AI が敵弾を予測回避しつつ最下段の敵から撃ち落とす。ウェーブを全滅させると
    レベルアップし、敵の数・速度・弾が増えてどんどん難しくなる。全滅すると Lv.1 から自動リスタート
  - **ボンバーマン** — 4 体のドット絵ロボット AI が自動で爆弾を置き合う 4 人対戦。爆風の到達時間を読んで退避し、
    逃げ道がある時だけ爆弾を設置。一定時間でブロックが外周から積まれて盤面が狭まる（サドンデス）。
    最後の 1 体になると次ラウンドを自動開始。AI 自動運転時のみ高速化、手動操作時は通常速度。
    フッターの −/＋ でブロック密度を調整、ウィジェットをドラッグで広げると盤面のマス数が増える（最大 17×19）
  - **テトリス** — AI が全配置（回転×位置）を評価して最善手を選ぶ（集約高さ・消去ライン・穴・凸凹で評価）。
    ライン消去でレベルが上がり落下が速くなる。ゴースト＆ NEXT 表示、積み切ると自動リスタート
  - **スネーク** — 大きめのドット絵盤面。AI は餌への最短路を取りつつ「進んだ先に十分な空間が残る時だけ」進み、
    危険なら最も広い空間へ逃げて自滅を避ける。餌を食べるほどレベルが上がり速くなる
  - **ライフゲーム** — Conway のライフゲーム（トーラス・ドット絵）。初期値はランダムで、停止・全滅・
    **周期1/2のループ（点滅で止まる状態）**・長く続く低活性を検出するとランダムに細胞を足して動きを復活。
    フッターのセレクトで有名プリセット（グライダー / グライダー銃 / パルサー / R-ペントミノ / どんぐり /
    LWSS / ペンタデカスロン）をセット可能。「＋ﾗﾝﾀﾞﾑ」ボタン（手動）でいつでもランダム追加
  - **ブロック崩し** — AI がパドルでボールを受け、全消しでレベルアップ（速く・多く・硬く）。
    開始レベルをフッターの −/＋ で指定でき、Lv5 以降はステージをランダム生成。
    アイテム（分裂 / ボール巨大化 / パドル拡大 / 残機+ / スロー）と、残像・グロー・パーティクル等のモダン演出付き
- **手動モード**: ウィジェットをクリックしてフォーカスすると AI が止まり、矢印キーで操作できます。
  フォーカスを外すと AI が再開（Rush Hour は車をクリックで選択 → 矢印キー）
- **「＋」ボタン**でウィジェットを複数並べられます（右下から左へ整列）
- **ゲーム選択**: ヘッダーのセレクトボックスで 7 ゲームから直接選べます
- **一括非表示と復帰**: ヘッダーの「—」ボタンで全ウィジェットを一括で隠せます。復帰は次のいずれかで:
  - グローバルショートカット **⌘/Ctrl + Shift + H**（取得済みでない場合は ⌥+H 等にフォールバック）
  - **メニューバーのトレイアイコン**（左クリックで復帰、メニューから「すべて表示」「新しいウィジェットを追加」「終了」）
  - **Dock アイコン**をクリック（macOS）
- **前面 / 背面の切替**: トレイメニューの「前面に表示」「背面に表示」、またはショートカット **⌘/Ctrl + Shift + B**。
  背面は「常に最前面」を解除して他ウィンドウの後ろ（デスクトップ寄り）に置く挙動です
- ウィンドウ端のドラッグでリサイズ（縦横比固定）、ヘッダーのドラッグで移動
- スコア・周回数・最大連鎖などは自動保存

## 開発者向け

```bash
git clone https://github.com/nmatsumoto4/desk-widgets.git
cd desk-widgets
npm install
npm start        # 起動
npm run dist     # 配布用ビルド（dist/ に出力）
```

### ファイル構成

| ファイル | 役割 |
|---|---|
| `main.js` / `preload.js` | Electron メイン（ウィンドウ生成・整列・IPC） |
| `index.html` / `style.css` / `app.js` | ゲームウィジェット共通シェル（モード切替・手動/AI 切替） |
| `game.js` / `ai.js` / `mod2048.js` | 2048 ロジック / Expectimax AI / 描画 |
| `puyo.js` / `puyo-ai.js` / `mod-puyo.js` | ぷよぷよ ロジック / 連鎖 AI / 描画 |
| `rush.js` / `mod-rush.js` | Rush Hour ロジック・BFS ソルバー・問題生成 / 描画 |
| `mod-invaders.js` | インベーダー（ゲーム進行・回避/掃討 AI・レベル管理・描画） |
| `mod-bomber.js` | ボンバーマン（4 体 AI 対戦・爆風危険マップ/退避 AI・ブロック調整・描画） |
| `mod-tetris.js` | テトリス（配置評価 AI・レベル/落下速度・ゴースト/NEXT・描画） |
| `mod-snake.js` | スネーク（最短路＋空間確保 AI・可変大グリッド・ドット絵描画） |
| `mod-life.js` | ライフゲーム（トーラス B3/S23・停滞検出ランダム注入・有名プリセット・ドット絵） |

新しいゲームは `show/hide/setAuto/key/relayout` の共通インターフェースに載せるだけで追加できます。
